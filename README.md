# 🩺 Diabetes Risk Prediction System

This project is a Machine Learning web application that predicts the risk of diabetes using patient medical data.

## 🚀 Live Demo
(To be added after deployment)

## 📊 Dataset
Pima Indians Diabetes Dataset

## 🧠 Model Used
Logistic Regression (with feature scaling)

## ⚙️ Features Used
- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age

## 🖥️ Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy

## ▶️ How to Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
